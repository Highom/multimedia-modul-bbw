Image formats

I use jpg for all images of products and the example Images. The Logo is a png file and the little controller Icon is a svg. Also the favicon is an animated ico

The JPG/JFIF format is used to store and transfer pictures across the web. It has lossy compression and does not support transparency.png has lossless compression. It supports a 24-bit color range and allows transparency. Gif only supports 256 colors but can store animations and transparency. Svgs are vector graphics. Instead of using pixels it uses elements wwith a magnitude and direction. This makes it indefinetly scalable
